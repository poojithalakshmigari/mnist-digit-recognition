# Handwritten Digit Recognition (MNIST)

## Project Overview
This project focuses on classifying handwritten digits (0–9) using the MNIST dataset. 
The goal is to build and compare multiple machine learning models to identify the best performing classifier.

## Dataset
- 70,000 grayscale images
- Image size: 28x28 pixels
- 10 classes (digits 0–9)

## Steps Performed
- Data preprocessing and normalization
- Train-test split
- Model training using Logistic Regression / SVM
- Model evaluation using accuracy and confusion matrix

## Results
Achieved high accuracy in digit classification after tuning model parameters.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow/Keras (if used)

## Conclusion
The project demonstrates image classification using machine learning and highlights the importance of preprocessing and model comparison.
